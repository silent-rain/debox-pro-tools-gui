# React Mobile 模板

## 技术栈

- [React](https://reactjs.org/)
- [Vite](https://vitejs.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Ant Design Mobile](https://mobile.ant.design/zh)
- [ESLint](https://eslint.org/)
- [Prettier](https://prettier.io/)
- Mockjs
- axios

## 开发与编译

```shell
# 安装依赖
pnpm install

# 开发模式
pnpm dev

# 编译
pnpm build
```
